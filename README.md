grunt-replace-status
====================

local, stage, production 으로 상태 분리하여 replace 등록가능


로컬 테스트 환경
================
  - replace-local
  - 실행 : grunt replace-local
  

스테이징 테스트 환경
====================
  - replace-stage
  - 실행 : grunt replace-stage


프로덕션 환경
=============
  - replace 
  - 실행 : grunt replace
  - 배포 : grunt build   (배포명령을 수행할 때 자동 수행함)
